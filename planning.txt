SET UP REPO
- create repo on github and select 'node' option for gitignore'
- clone repo to local machine

SET UP DB
- create database in psql "CREATE DATABASE database_name OWNER dev_user"

SET UP EXPRESS APP
- run 'npm init -y' to create package.json, config file
- run 'npm install express' to install express
- create app.js file
- run from script, insert '    "start": "node --watch app.js" in the script of package and use 'npm start' to run your app

SET UP PRISMA
- install prisma with the modified instructions from Odin
https://www.theodinproject.com/lessons/nodejs-prisma-orm
- installs prisma, prisma client, pg postgres driver and dotenv for loading env variables
